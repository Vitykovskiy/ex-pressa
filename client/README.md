# Клиент Ex-Pressa

Фронтенд построен на Vue 3, Vite и Vuetify. Проект используется внутри монорепозитория, поэтому устанавливать зависимости и запускать скрипты рекомендуется из корня (npm install, npm run client:dev).

### Полезные команды
- npm run client:dev — дев-сервер Vite с горячей перезагрузкой.
- npm run client:build — production-сборка.
- npm run client:lint — проверка ESLint.

Перед стартом убедитесь, что заполнены адреса API и фронтенда в корневом .env.
